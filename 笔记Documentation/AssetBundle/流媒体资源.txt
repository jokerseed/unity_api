Unity中的大多数资源在构建时都会合并到项目中。但是，将文件放入目标计算机上的普通文件系统以使其可以通过路径名访问有时很有用。
如在IOS设备上部署电影文件；原始电影文件必须位于文件系统中的某个位置以便有PlayMovie函数进行播放。

放置在Unity项目中名为StreamingAssets的文件夹中的所有文件都将逐字复制到目标计算机上的特定文件夹。可使用
Application.streamingAssetPath属性来检索此文件夹。
在任何情况下，最好使用Application.streamingAssetsPath来获取StringAssets文件夹的位置。

此文件夹因平台而异同
在桌面计算机（Mac OS 或 Windows）上，可使用以下代码获取文件的位置：
path = Application.dataPath + "/StreamingAssets";

在 iOS 上，使用：
path = Application.dataPath + "/Raw";

在 Android 上，使用：
path = "jar:file://" + Application.dataPath + "!/assets/";
